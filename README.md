# Aim Training Game with Python

First project using Pygame module in Python.

## Dependencies
* Python 3
* Pygame module

## Installation
1. Clone repo:
``` git clone https://github.com/ManuLosta/aim-training.git
```

2. If Pygame isn't installed:
``` pip install pygame
```
